🧠 SICKLE CELL DISEASE PREDICTION SYSTEM

AI-powered clinical decision support system for predicting sickle cell complications and risk levels.

🚀 PROJECT OVERVIEW

This project is an intelligent healthcare system designed to predict complications in patients with Sickle Cell Disease (SCD) using machine learning.

It analyzes clinical parameters such as hemoglobin levels, blood counts, and patient symptoms to provide:

Early risk prediction
Real-time insights
Clinical recommendations

The system helps doctors and caregivers take proactive decisions and improve patient outcomes.

✨ FEATURES
🧠 AI-based complication prediction
📊 Interactive dashboard with analytics
📋 Patient history tracking
⚠️ Risk level classification (LOW / MODERATE / HIGH)
💡 Clinical suggestions based on predictions
🔄 Real-time data processing

⚙️ HOW TO RUN
# Clone the repository
git clone https://github.com/your-username/sickle-cell-prediction.git

# Navigate to project folder
cd sickle-cell-prediction

# Install dependencies
pip install flask numpy pandas scikit-learn

# Run the application
python app.py

Open your browser and go to:
👉 http://127.0.0.1:5000/
